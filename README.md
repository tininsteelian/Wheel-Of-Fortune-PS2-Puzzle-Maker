# Wheel of Fortune PS2 Puzzle Maker
This command-line python program lets you create custom puzzles for the PS2 version of Wheel of Fortune.

## Usage
1. Make a custom puzzle file (see below).
2. Extract the contents of the Wheel of Fortune PS2 iso using 7zip (or your preferred tool).
3. Run the program in terminal with the following parameters:
```
WheelOfFortunePS2PuzzleMaker.py [input file] [output folder]
```
4. If the output folder was not set to the extracted game's DATA folder, replace the PUZZLES.BIN and/or PUZZLEB.BIN files in that folder with the ones generated by the program.
5. Create a new iso of the game using ImgBurn (or your preferred tool) by selecting "Create image file from files/folders", then adding the extracted game's root folder as the source.

## Making a Custom Puzzle File
To use the puzzle maker, you'll need to provide a custom puzzle file. This is a text file where each line is either a category or a puzzle.

### Puzzles
Each line of text is a unique puzzle. You can manually separate the puzzle into rows by typing '|' between the rows, or let the program do this for you. Keep in mind that the puzzle has to fit on the Wheel of Fortune board, which can hold 12 characters on the first and fourth rows, and 14 characters on the second and third rows. Puzzles using one or two rows start on the second row, so you only need to worry about the rows that hold 12 characters if the puzzle uses 3 or 4 rows.

Only certain characters are supported by the game. These include `A-Z`, as well as the special characters `#&-'.?`. Numbers are not supported, nor are any other special characters. Capitalization does not matter.

Here's an example of some valid puzzles:
```
TEST
NINTENDO WII U
GREEN EGGS|AND HAM
SALAMANDER COUNTY PUBLIC TELEVISION
SALAMANDER|COUNTY|PUBLIC|TELEVISION
```

By default, the program will generate normal round puzzles (PUZZLES.BIN). If you want to generate bonus round puzzles (PUZZLEB.BIN), make the first line of the custom puzzles file `!bonus` .

The base game has 2499 normal puzzles and 500 bonus puzzles. You can have more than 2499 puzzles in your custom puzzle file, but the game may freeze for some time (1-2 minutes) before the start of a new round if you do.

### Categories
To set a category, type a `!` at the start of the line, followed by the name of the category. Every puzzle in subsequent lines will use that category until a new category is added. If you don't set any categories, 'PHRASE' will be used as a default.

Here is a list of categories you can use:
```
!phrase
!person
!people
!title
!landmark
!place
!thing
!quotation
!event
!fictional character
!star & role
!occupation
!husband & wife
!same name
!before & after
!family
!artist / song
!title / author
!the 60's
!nickname
!classic tv
!fictional characters
!proper name
!show biz
!the 70's
!the 80's
!the 90's
!things
!author / title
!around the house
!on the map
!rhyme time
!headline
```

These categories have a multiple-choice question that appears after the puzzle is solved, with three possible answers:
```
!clue
!fill in the blank
!next line please
!slogan
!where are we?
!who is it?
!who said it?
!fill in the number
```
The multiple-choice question is required for any puzzles in these categories. To add it, type a `~` after the puzzle, followed by the three possible answers, each of which are separated by a `/`. Put a `*` in front of the correct answer. Here's an example:
```
PICK THE|RIGHT ANSWER~WRONG/ALSO WRONG/*CORRECT
```

Additionally, there are several categories that aren't used by any puzzles in the game normally, but are still accessible and can be used:
```
!fictional place
!places
!person / title
!foreign word
!slang
!proper names
!song / artist
!events
!where are they?
!puzzler
!surprise question
!person or people
```
Some of these categories have bugs: 'PLACES' is erroneously displayed as 'PLACE', and 'WHERE ARE THEY?', 'PUZZLER', 'SURPRISE QUESTION', and 'PERSON OR PEOPLE' all play an incorrect voice clip ('HEADLINE' also has this bug, but that category is actually used).

### Full Example
Here's an example of a complete custom puzzle file:
```
!thing
A BOX OF|JOLLY|RANCHER|CHEWS
SEGA GENESIS|CARTRIDGE
TWO THOUSAND|AND SIX|HONDA CIVIC
!fictional character
TACO TOM
!who said it?
FOUR SCORE|AND SEVEN|YEARS AGO~*ABRAHAM LINCOLN/ALBERT EINSTEIN/WAYNE GRETZKY
THAT'S BEANS~EMBER/*GEOFFREY/RICH RICHARDSON
```

## Known Issues
Sometimes the game will generate a blank puzzle with the category 'PHRASE'. When this happens, the game will crash when a human player attempts to solve it, although computer players can solve it without the game crashing. I think there are two reasons this can happen: 
- the game tries to avoid picking a puzzle with the same category as the previous round's puzzle
- the game likely marks enough puzzles as recently played (so that the same puzzle doesn't get repeated too quickly) that it runs out of "fresh" puzzles to choose from

Using a custom puzzle file with multiple categories and a large number of puzzles will mitigate this issue. You can copy and paste the same puzzles multiple times in the custom puzzles file, although you then run the risk of getting the same puzzle multiple times in the game. You can also try deleting the game's save file if the issue happens repeatedly.
